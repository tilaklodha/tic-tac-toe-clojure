# tic-tac-toe

Tic Tac Toe in clojure

Give option to play:

  - human vs human
  - human vs computer
  - computer vs human
  - computer vs computer


## Installation

- Requires Java

### Setup leiningen

```
$ brew install leiningen
```


## Publish jar

```
$ lein uberjar
```

## Install dependencies

```
$ lein deps
```

## Run Game

`lein run`

## Run tests

`lein test`

## Code-coverage
`lein cloverage`
`open target/coverage/index.html`
